this code will turn your mobile phone camera to a webcam
with WI-FI using openCV.

First : Your mobile phone and your camputer should
conect to the same wifi network.

Second : Download the andriod version of DroidCam.
Open it and see your ip address.

Third : Put your address in the following format :
http://192.x.x.x:xxxx/video
e.g. : http://192.168.1.8:8080/video

Four : Open the app on your phone and run the code. 

Note : Watch the provided video to see the project performance
